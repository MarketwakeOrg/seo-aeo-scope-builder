# Marketwake SEO + AEO Scope Builder

Internal sales tool for building and presenting SEO/AEO engagement scopes in real time. Single self-contained HTML file, no build step, no frameworks.

## Usage

Open `scope-builder.html` in any modern browser.

1. Pick a base package (Undercurrent / Current / High Tide / Rip Tide)
2. Layer on add-ons (briefs, authored blogs, on-page opts, AEO tier, Yext, QBR)
3. Walk the client through the live-updating scope panel
4. Click **Request this scope** to print or save as PDF, or **email this scope** to draft a plain-text summary

## Pricing Logic

- Sliders for briefs / authored blogs / on-page opts show the TOTAL quantity. Quantities bundled into the selected package are $0, only the excess is charged.
- Every authored blog ships with its own brief, so authored count offsets briefs charges.
- Briefs follow the same 4 standard + 1 long-form cadence per 5 as authored blogs.
- Volume breaks drop unit prices every 5 units for briefs, authored, and on-page opts.
